# Загрузчик медиафайлов

Приложение для скачивания аудио и видео файлов с различных сайтов с графическим интерфейсом.

## Возможности

- Скачивание видео в формате MP4
- Скачивание аудио в формате MP3
- Поддержка различных платформ (YouTube, Vimeo, и др.)
- Графический интерейс пользователя
- Отображение прогресса загрузки
- Выбор директории сохранения файлов

## Требования

- Python 3.8 или выше
- Интернет соединение
- FFmpeg (для конвертации в MP3)

## Установка

1. Установите Python 3.8 или выше с [официального сайта](https://www.python.org/downloads/)

2. Установите FFmpeg:
   - **Windows**: Скачайте с [https://ffmpeg.org/download.html](https://ffmpeg.org/download.html)
   - **macOS**: `brew install ffmpeg`
   - **Linux**: `sudo apt install ffmpeg` (Ubuntu/Debian) или эквивалент для вашего дистрибутива

3. Установите зависимости:
   ```bash
   pip install yt-dlp
   ```

## Установка альт
 - Скачать .exe файл их репризитория

## Использование

### Вариант 1: Через терминал
```bash
python main.py
```

### Вариант 2: Использование исполняемых файлов
- **Windows**: Запустите `run_app.bat` или `run_app.ps1`
- **Linux/macOS**: Запустите `run_app.sh`

### Вариант 3: Через exe
 - Запустить exe файл

После запуска:
1. Введите URL видео в поле ввода
2. Выберите формат (MP4 или MP3)
3. Выберите директорию для сохранения с помощью кнопки "Обзор..."
4. Нажмите кнопку "Загрузить"

## Поддерживаемые сайты

Приложение поддерживает большинство сайтов, поддерживаемых yt-dlp:
- YouTube
- Vimeo
- SoundCloud
- И многие другие

Полный список поддерживаемых сайтов см. в документации [yt-dlp](https://github.com/yt-dlp/yt-dlp/blob/master/supportedsites.md)

## Решение проблем

### Если приложение не запускается:

1. Убедитесь, что установлен Python 3.8 или выше
2. Проверьте, что все зависимости установлены:
   ```bash
   pip install yt-dlp
   ```

### Если загрузка не работает:

1. Проверьте URL - он должен быть прямой ссылкой на видео
2. Убедитесь, что у вас есть доступ к интернету
3. Проверьте, что сайт поддерживается yt-dlp

### Если конвертация в MP3 не работает:

1. Убедитесь, что FFmpeg установлен и доступен в системе
2. Проверьте, что путь к FFmpeg добавлен в переменные среды

Это приложение предоставляется "как есть" без каких-либо гарантий. Убедитесь, что вы имеете право скачивать контент с сайтов, которые вы используете.
